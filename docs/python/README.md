## Quick Start

1. Set up the environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

2. Copy the template:
```bash
cp src/python/template.py my_script.py
```

3. Run the script:
```bash
python my_script.py [options]
```
